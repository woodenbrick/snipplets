h2. Snipplets

A cross platform databank to store snippets of data such as
* code (with syntax highlighting)
* gpg keys, with the option to import them into apt

Snipplets are importable/exportable as XML. Future releases will including tagging, search and encryption.

The program can be run from source: ./bin/snipplets 
or installed using: python setup.py install (as superuser)
There is also a "PPA repository for Ubuntu/Debian users":https://launchpad.net/~snipplets/+archive/ppa

A windows installer coming soon.
